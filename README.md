# Portfolio Project

## Overview

This project is a professional portfolio website built to showcase frontend and backend development skills, project work, and qualifications. It serves as a polished personal branding platform designed for hiring managers and recruiters, highlighting technical expertise, service offerings, and contact information in a clean, modern layout.

## Features

- Responsive, mobile-first design
- Smooth navigation with section-based scrolling
- Interactive project showcase using Swiper slider
- Dark mode toggle and scroll-to-top support
- Contact section with form layout and direct messaging links

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Swiper.js for project carousel
- EmailJS integration for form messaging support

## Purpose

This portfolio demonstrates the ability to build a complete web experience from layout and styling to interactive behavior. It is intended for use in job applications, client outreach, and professional networking.

## Project Structure

- `index.html` — main landing page and portfolio content
- `assets/css/style.css` — custom styling and responsive layout
- `assets/js/main.js` — client-side interaction logic
- `assets/img/` — portfolio visuals, icons, and graphic assets
- `assets/docs/` — supporting documents such as resume PDF

## Contact

Designed to support direct contact via email and social profiles, this project positions the creator as a well-rounded web developer ready to deliver modern digital experiences.
